# Chatroom

## 运行步骤
```
$ virtualenv env              # 新建一个虚拟环境
$ env\Scripts\activate        # 激活虚拟环境
$ pip install -r requirements # 生成文件需要的包
$ flask forge                 # 生成虚假数据
$ flask run/python start.py   # 运行
* Running on http://127.0.0.1:5000/
```

## 调试模式
```
$ $env:FLASK_DEBUG = "1" or
$ set FLASK_DEBUG = "1"
```
